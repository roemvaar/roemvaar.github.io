BeagleV Board Bring-Up with Yocto

More information at https://beagleboard.org/beaglev

In this lab, I'll introduce Yocto and Poky and show how to use it and show how to use it to build an embedded Linux image for BeagleV - a development kit that has a RISC-V SoC.

Lab Goals: - Serial communication with BeagleV board Introduction: A Linux Based Embedded System has six main components: 1. Bootloader 2. Linux Kernel 3. Device Tree 4. Root Filesystem 5. System Programs 6. Application The Yocto Project "It's not an embedded Linux distribution, it creates a custom one for you." Poky is a reference distribution of the Yocto Project which contain the openEmbedded build system and some meta data that helps people to build their own distribution of Linux.
